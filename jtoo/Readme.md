A Rust library for serializing and deserializing the [JTOO data format](https://github.com/mleonhard/jtoo-format).

# Features
- `forbid(unsafe_code)`
- No dependencies
- Good test coverage (NN%)

# Limitations
- New, not proven in production.
- To do:
    - Pack trait
    - Unpack trait
